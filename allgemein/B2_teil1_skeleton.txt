B2 Sprechen 

TEIL - I : In Teil 1 halten Sie einen kurzen Vortrag und sprechen mit Ihrer Gesprächspartnerin/Ihrem Gesprächspartner darüber. 
Wählen Sie dafür ein Thema (1 oder 2) aus (circa 4 Minuten).

Zeit : circa 4 Minuten

Points :
----------
* task completeness
* sentence connection to other sentences (cohesion)
* Discussion amongst partners
* wortschatz (voacabulary)
* Structure
* Pronunciation

Information : http://bfu.goethe.de/b2_mod_2MX6/sprechen.php

============
Beschreibung
============
* Beschreiben Sie mehrere Alternativen. ( describe all alternatives, points, methods)
* Nennen Sie Vor- und Nachteile und bewerten Sie diese. (list advantage and disadvantage of these alternatives, methods, points)
* Beschreiben Sie eine Möglichkeit genauer. ( describe one altiernative, option exactly)

======
Verbs
======
* beschreieben ( descirbe)
* erklären (clarify) => when you have to explain something in more detail
* erläutern ( pronunciation ERLOYTERN)  => like describe, 
* betonen (kannada : ottu kodu, habe betont) => Ich möchte betonen, dass wir kein Plastik mehr benutzen sollen.
* arguementieren ( to argue) => ich möchte argumentieren, dass CO2 Emissionen gefährlich für Natur ist.
(Co2=Kohlendioxide, Kohlenstoff in German means Carbon, Kohle (English Coal, Kannada : Kalliddalu) which proudces carbon after burning.
* schildern (to describe) comes from the word "Shild" => Falaka, Board
* vergleichen ( to compare)
* vorlegen ( to present, nimmma munde iduvudu, vor-munde, legen-idu)
* äußern (vyaktapadisu, express)
* ausdrucken (vyaktapadisu, express)
